# Random Quote Generator (Python)

A lightweight CLI application that prints a random quote, with an option to filter by category (tag).

## Features
- Print a random quote from a built-in collection
- Filter quotes by tag (e.g., `programming`, `growth`, `discipline`)
- List available tags
- Simple menu-driven CLI

## Tech Stack
- Python 3 (standard library only)

## Run Locally
```bash
python src/main.py
